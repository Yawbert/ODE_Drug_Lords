Numerical Methods

Because a closed-form analytical solution is intractable for the full coupled system, we deploy numerical integration techniques. We implemented three approaches to compare accuracy and stability.

Euler Method (Forward)

A first-order method. Simple to implement, but highly susceptible to instability for stiff equations unless the step size h is exceedingly small.

function [T, Y] = forward_euler(ode_fun, t_span, y0, h) t = t_span(1):h:t_span(2); n = length(t); y = zeros(length(y0), n); y(:,1) = y0; for i = 1:(n-1) y(:,i+1) = y(:,i) + h * ode_fun(t(i), y(:,i)); end T = t'; Y = y'; end
Runge-Kutta 4th Order (RK4)

A robust 4th-order method that evaluates the derivative at four points per step. Offers an excellent balance of computational cost and accuracy for our system.

function [T, Y] = rk4(ode_fun, t_span, y0, h) t = t_span(1):h:t_span(2); n = length(t); y = zeros(length(y0), n); y(:,1) = y0; for i = 1:(n-1) k1 = ode_fun(t(i), y(:,i)); k2 = ode_fun(t(i) + h/2, y(:,i) + h/2 * k1); k3 = ode_fun(t(i) + h/2, y(:,i) + h/2 * k2); k4 = ode_fun(t(i) + h, y(:,i) + h * k3); y(:,i+1) = y(:,i) + (h/6) * (k1 + 2*k2 + 2*k3 + k4); end T = t'; Y = y'; end
MATLAB ode45 (Adaptive Solver)

As a benchmark, we use MATLAB's built-in ode45, which is based on an explicit Runge-Kutta (4,5) formula — the Dormand-Prince pair. It uses adaptive step sizing to keep the local error within specified tolerances (RelTol and AbsTol). Our manual RK4 and Euler implementations are compared against ode45 output to assess error accumulation.
